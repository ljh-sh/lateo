# v0.0.1 — scaffold

## Headline

**lateo repo bootstrapped.** A single Rust crate: `src/lib.rs` (engine stub
exposing `Op`) + `src/main.rs` (CLI dispatch over `hide` / `extract` / `mark` /
`verify` / `probe`). Governance, lean 2-stage CI, and reproducible release
pipeline follow the rust-oss skill as adopted by upxz / roff.

## What this is

lateo is an imperceptible data-embedding toolkit for images, covering two
complementary primitives:

- **steganography** — hide a covert *message* (existence-denied, fragile);
- **watermarking** — hide a robust *imprint* (existence-asserted, survives
  re-encoding).

The name is Latin *lateo* — "I lie hidden".

## Compatibility

- No public embedding API yet (engines unimplemented).
- `cargo build` / `cargo test` green on a fresh clone.

## Install

Not yet published. Build from source:

```bash
cargo build --release
./target/release/lateo --help
```
